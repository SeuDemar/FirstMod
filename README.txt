Projeto criado como forma de tutorial de aprendizado para utilizar o forge como ferramenta de desenvolvimento de mods para o minecraft
--------------------------------------------------------------------------------------------------------------------------------------
Utilizando :
1 - VScode configurado com as extensões necessárias para java
2 - Graddle
3 - Forge mdk 1.20.1
4 - Minecraft 1.20.1
--------------------------------------------------------------------------------------------------------------------------------------
Arquivos vindos majoritariamente do MDK.
Links das ferramentas utilizadas serão postadas futuramente.
